## 🚀 Project Overview
**Annadata Connect** is a high-performance, enterprise-grade digital food distribution platform designed to bridge the gap between India's food surplus (hotels, weddings, businesses) and the high-need underserved clusters. Built with a **"Soft Slate" Dark Aesthetics** and **Blockchain-Inspired Transparency**, it ensures that every single meal is tracked, verified, and delivered with absolute dignity.

### 🔱 The 25-Feature Stakeholder Ecosystem
The platform delivers a comprehensive "Beast" mode experience for 5 distinct user roles:

#### 1. 🏨 Bulk Donor Dashboard
*   **AI Smart Scheduling**: Automated pickup slots based on meal type.
*   **80G/CSR Vault**: Instant blockchain-verified tax exemption receipts.
*   **Carbon ESG Analyst**: Live tracking of CO2 savings & environmental impact.
*   **Urgent Perishables Toggle**: Express routing for food with <2hrs shelf life.
*   **Global Leaderboard**: Gamified rankings for top corporate donors.

#### 2. 🛡️ NGO Operations Hub
*   **Predictive Storage Planner**: AI-calculated shelf-life & warehouse capacity.
*   **Volunteer Dispatch Node**: Dynamic matching of nearby last-mile vectors.
*   **Digital Hygiene Gates**: Structured verification logs for thermal/food quality.
*   **CSR Funding Tracker**: Live dashboard for institutional grant utilization.
*   **Real-time Distribution Feed**: Live ticker of community impact.

#### 3. 🗺️ District Admin (Govt)
*   **AI Ghost-Hunting Predictor**: Neural detection of supply-chain diversion.
*   **Mass-SMS NIC Broadcaster**: Instant ping to thousands of beneficiaries.
*   **FPS License Revocation**: One-tap ban on corrupt distribution points.
*   **PM-GKAY Budget Mapper**: Macro-level visibility on national funding flow.
*   **Inclusion Algorithm**: Automated Aadhaar-driven error correction.

#### 4. 🚴 Volunteer Mobile App
*   **Neural Proof of Delivery**: Camera AI verification for authenticated handoffs.
*   **Secure Offline Vault**: Functionality in low-network connectivity slums.
*   **Impact Coin Wallet**: Gamified rewards for successful delivery vectors.
*   **Hyper-local Route Sync**: Advanced navigation overlay for <2km radius.
*   **SOS Broadcaster**: Live GPS/Mic signal broadcast for field safety.

#### 5. 🏠 Beneficiary Portal
*   **Aadhaar-Linked Scheduler**: Dignified, queue-free slot booking.
*   **Family Portability (ONORC)**: Seamless registry migration for migrants.
*   **Blockchain Grievance**: Anonymous reporting of corruption or quality issues.
*   **Nutrition Analyst**: Family protein/calorie tracking via ration history.
*   **Family Registry Management**: Direct Aadhaar-linked dependent additions.

## 🛠️ Tech Stack & Architecture
- **Framework**: [React.js](https://reactjs.org/) + [Vite](https://vitejs.dev/) (Ultra-fast HMR)
- **Styling**: Vanilla CSS with **Glassmorphism**, **Neon Utilities**, and **Soft Slate Aesthetics**.
- **3D Visualization**: [Three.js](https://threejs.org/) (Interactive Network Constellation Hero & 3D Impact Orb).
- **Data Analytics**: [Recharts](https://recharts.org/) (Environmental Delta & Nutrition mapping).
- **State Management**: [Zustand](https://github.com/pmndrs/zustand) (with `localStorage` persistence).
- **Icons**: [Lucide-React](https://lucide.dev/) (Enterprise iconography).
- **Animations**: CSS3 Cubic-Bezier transitions & keyframe-based "Beast-mode" sequences.

## 🛠️ Installation & Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/iamshubham369/ThinkTank_devsprint.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development environment:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
